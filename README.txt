This is version 0.1 of "liblayout", an experimental 2D layout library.

Of note:

a) The license is the Lesser General Public License (LPGL), see lpgl.txt for the 
full license.  

b) The language is C (ANSI C99), for ease of linking with other languages.

c) Code documentation has fairly high coverage, and is generated automatically 
from the code by Doxygen (http://www.stack.nl/~dimitri/doxygen/).
