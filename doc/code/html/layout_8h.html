void lay_register_rects ( lay_statep  state,
lay_coord_t *  rect_pos,
const ptrdiff_t  pos_skip,
lay_extent_t *  rect_size,
const ptrdiff_t  size_skip,
const int  count 
)

Register the positions and sizes of a set of rectangles with liblayout.

Parameters
stateThe internal state structure.
rect_posA pointer to the x-coordinate of the first size. The y-coordinate of the first size is assumed to exist at *(rect_pos + 1). The pointer can be NULL, but no operations requiring sizes may be used.
pos_skipThe number of bytes to add to rect_pos to get to the x-coordinate of the next size. If zero, then the data is assumed to be tightly packed, and pos_skip will be set to 2 * sizeof(lay_coord_t). Can be negative.
rect_sizeA pointer to the width of the first size. The height of the first size is assumed to exist at *(p + 1). The pointer can be NULL, but no operations requiring sizes may be used.
size_skipThe number of bytes to add to p to get to the width of the next size. If zero, then the data is assumed to be tightly packed, and skip will be set to 2 * sizeof(lay_extent_t). Can be negative.
countThe number of rectangles.
